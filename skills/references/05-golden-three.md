# 黄金三章 · 深度方法论

---

## 模块目标

用 3 章锁住读者，每一章都有明确的钩子和节奏。

## 5 层追问链（详细版）

### 第 1 层：第一章第一段读者看到什么？

**问题模板**：
- 第一章第一段，读者看到什么？
- 有什么冲突/反常/悬念让他们继续？

**9 种开篇钩子**：
1. **悬念钩**："我从未想过，自己会在 24 岁生日那天杀掉最好的朋友"
2. **反常钩**："当所有人都在逃跑时，只有我朝火里走去"
3. **预言钩**："所有人都说我会成为英雄，但我知道自己会成为怪物"
4. **对比钩**："三年前的今天我是废物，三年后……"
5. **冲突钩**："剑刺入她胸口的那一刻，我才意识到她是我母亲"
6. **行动钩**："我没有犹豫，从三十楼跳了下去"
7. **发现钩**："那封信上写着我已经死了三年的名字"
8. **决定钩**："我决定在今天结束一切"
9. **反差钩**："外卖小哥单手接住了那颗子弹"

### 第 2 层：主角的转变

**问题模板**：
- 主角在前三章完成了什么转变？
- 从「谁」变成了「谁」？
- 这个转变是否清晰可见？

**关键测试**：读者读完 3 章能说出"主角变了"吗？

### 第 3 层：金手指揭示

**问题模板**：
- 金手指/超能力在第几章揭示？
- 揭示方式是「获得」还是「被发现」？
- 揭示时的爽点场景是什么？

**两种揭示方式**：
- **获得型**：主角主动获取（拜师、拾物、传承）
- **发现型**：主角发现自己本来就有（血脉觉醒、记忆恢复）

**关键洞察**：金手指揭示的"时机"比"方式"更重要。

### 第 4 层：上架钩

**问题模板**：
- 第三章结尾的"上架钩"是什么？
- 读者为什么必须看第四章？

**常见上架钩**：
- 反派出场："真正的噩梦，从他出现那一刻开始"
- 转折点："我以为这只是开始，但命运告诉我这是结束"
- 危险迫近："三天后，整个大陆都会知道我的名字"
- 选择："我必须在两个选择中选一个，而每一个都是地狱"

### 第 5 层：300 字开篇

**问题模板**：
- 如果你只能用 300 字写开篇，你会从哪个画面开始？

**关键测试**：作者能用 300 字说清第一章第一段的内容吗？

## 关键方法论

### 冲突前置
- 第一个字就要有冲突
- 反例：用 500 字介绍世界观
- 正例：主角被追杀 / 收到神秘信 / 醒来发现自己在异世界

### 人设反差
- 普通身份 + 超常秘密
- 表面是书店老板，真实身份是退役杀手
- 让读者带着"窥探欲"读下去

### 三屏节奏
- 每三屏（~1000-1500 字）必须一个变化
- 变化类型：场景切换 / 冲突升级 / 信息揭示 / 情绪转折

### 9 种开篇钩子（详）

每种钩子的适用场景：

| 钩子 | 适用 | 不适用 |
|------|------|--------|
| 悬念钩 | 所有类型 | - |
| 反常钩 | 都市/玄幻 | 纯爽文 |
| 预言钩 | 玄幻/奇幻 | 都市现实 |
| 对比钩 | 升级流 | 反差不够的话剧 |
| 冲突钩 | 动作类 | 慢热文 |
| 行动钩 | 动作类 | 慢热文 |
| 发现钩 | 悬疑/推理 | 升级流 |
| 决定钩 | 严肃向 | 纯爽文 |
| 反差钩 | 反差萌类 | - |

## 反模式警告

| 反模式 | 触发信号 | 警告语 |
|--------|---------|--------|
| 开篇铺设定 | 第 1 章用 500+ 字解释世界观 | 你的设定很好，但读者想看故事 |
| 无冲突 | 第 1 章第 1 段主角在吃饭/睡觉 | 网文读者 3 章内看不到冲突，80% 会弃书 |
| 人设平庸 | 主角"普通人+努力"无特殊点 | 给主角一个秘密或反差 |
| 节奏拖沓 | 3 章讲完一段历史 | 3 章必须有：钩子 + 冲突 + 转变 + 上架钩 |
| 金手指过早 | 第 1 章就展示全部能力 | 让金手指慢慢揭示，吊读者胃口 |

## SB 字段定义（golden_three）

```typescript
golden_three: {
  chapter_1: {
    opening_hook: string;
    protagonist_intro: string;
    golden_finger_hint: string;
    chapter_end_hook: string;
  };
  chapter_2: {
    setting_expansion: string;
    conflict_escalation: string;
    golden_finger_reveal: string;
    chapter_end_hook: string;
  };
  chapter_3: {
    mini_climax: string;
    direction_lock: string;       // 方向锁定
    shelf_hook: string;            // 上架钩
    chapter_end_hook: string;
  };
  hooks_selected: string[];        // 选中的钩子类型
  screen_rhythm: string;           // 三屏节奏说明
}
```

## 完成标准

- 3 章的 hook / conflict / escalation 全有
- 第 1 章第 1 段钩子确定
- 第 3 章上架钩确定
- 三屏节奏说明清晰

## 推荐问题

1. "我想设计开篇" → 进入 ⑤ 黄金三章
2. "第 1 章怎么写？" → 直接进入第 1 层追问
3. "我的开篇太平" → 触发反模式警告 + 推荐 9 种钩子
4. "金手指何时揭示？" → 进入第 3 层追问
