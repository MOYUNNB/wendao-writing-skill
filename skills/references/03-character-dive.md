# 角色深潜 · 深度方法论

---

## 模块目标

创造有欲望、有缺陷、有执念的角色，而非纸片人。

## 5 层追问链（详细版）

### 第 1 层（表面）：最想让什么发生？

**问题模板**：
- 这个角色最想让什么发生？
- 他的显性欲望是什么？
- 用一句话说清

**关键测试**：显性欲望必须具体、可视化、可被读者记住。

### 第 2 层（动机）：为什么想要？

**问题模板**：
- 这个欲望的根源是什么经历？
- 这个经历发生在角色几岁？
- 这个经历如何塑造了他？

**关键洞察**：动机的根源往往是"创伤事件"或"关键转折"。

### 第 3 层（矛盾）：想要 vs 需要

**问题模板**：
- 他想要什么 vs 他真正需要什么？这两个一样吗？
- 如果不一样，差距是什么？

**McKee 核心洞见**：
- **Conscious Desire**：角色意识到的欲望（剧情线推动）
- **Unconscious Need**：角色真正需要但不自知的（角色弧光）
- **戏剧张力**：两者不同 → 故事动力

**示例**：
| 角色 | 显性欲望 | 真实需求 |
|------|---------|---------|
| 哈姆雷特 | 复仇 | 接受父亲的离去 |
| 林黛玉 | 与宝玉在一起 | 学会独立 |
| 你的主角 | ? | ? |

### 第 4 层（极限）：压力下的选择

**问题模板**：
- 如果保护这个欲望需要背叛最重要的人，他会怎么选？
- 如果救这个人需要放弃目标，他会怎么选？

**McKee 核心**：
- "True character is revealed only under pressure"
- 平静时的角色是"characterization"（人设）
- 压力下的角色是"true character"（真实性格）

**关键测试**：作者必须能说清"主角在压力下会怎么选"。

### 第 5 层（代价）：致命弱点

**问题模板**：
- 他的致命弱点是什么？
- 反派会如何利用它？
- 主角如何克服（或被弱点打败）？

**反模式警告**：
- ❌ "他就是强" → 无弱点 = 无成长空间
- ❌ "他有点骄傲" → 太泛
- ✅ "他无法放弃控制欲，每次胜利都必须按他的剧本走" → 具体

## 关键方法论

### 角色三大杀招

| 杀招 | 定义 | 例子 |
|------|------|------|
| **欲望 × 缺陷** | 显性欲望与致命缺陷的冲突 | 复仇 × 优柔寡断 |
| **能力 × 代价** | 金手指与使用代价 | 逆天功法 × 寿元减十年 |
| **人设 × 选择** | 表面性格与压力下真实选择的反差 | 表面冷酷 × 关键时刻救人 |

### 角色层级（金字塔）

| 层级 | 数量 | 引导深度 |
|------|------|---------|
| **Tier 1**（主角） | 1 | 全量深追（5 层） |
| **Tier 2**（重要配角） | 3-5 | 追一梗（一句话人设） |
| **Tier 3**（背景角色） | 无数 | 只追标签 |

### 一句话人设公式

```
[姓名] 是 [身份]，但 [反差/秘密]。
他最想要 [欲望]，但 [致命弱点] 让他 [困境]。
```

**示例**：
- 林风是废柴少年，但偶获上古传承。
- 他最想让家族复兴，但控制欲让他无法信任任何人。

## 反派设计

### 反派≠纸片

**反派的 3 个必备**：
1. **有动机**：为什么这样做？
2. **有立场**：相信什么？
3. **有逻辑**：行动是否自洽？

**反派的 4 层结构**：
- 表面：可恨的行为
- 动机：为什么会这样
- 立场：他相信的"正确"
- 软肋：他也有弱点

### 反派的常见错误

| 错误 | 例子 | 修正 |
|------|------|------|
| 纯粹恶 | "他就是坏" | 让他有自己的"正确" |
| 工具人 | "他专门用来被打脸" | 给他独立的目标 |
| 弱智 | "他明知会输还冲" | 让他有合理判断 |
| 无存在感 | "他没什么戏份" | 增加他的"主动行动" |

## 金手指设计

### 必须有代价

**金手指公式**：能力 + 代价 = 张力

| 代价类型 | 例子 |
|---------|------|
| **寿命** | 每用一次寿元减十年 |
| **情感** | 每用一次失去一段记忆 |
| **健康** | 长期使用会导致身体崩溃 |
| **关系** | 获得力量后会失去爱人的陪伴 |
| **道德** | 每次使用都需要做违心的事 |
| **轮回** | 每次重生都会忘记前世的羁绊 |

### 反模式

❌ **金手指无代价**：主角获得逆天能力，无任何限制 → 读者无法紧张
❌ **代价无表现**：设定了代价但剧情中从不体现 → 形同虚设
❌ **代价失控**：代价太大导致主角不愿使用 → 削弱主角

## SB 字段定义（characters）

```typescript
characters: {
  name: string;
  role: 'protagonist' | 'antagonist' | 'ally' | 'mentor' | 'rival';
  surface: {
    age: number;
    appearance: string;
    personality_tags: string[];
    occupation: string;
  };
  depth: {
    conscious_desire: string;
    unconscious_need: string;
    fatal_flaw: string;
    core_wound: string;          // 核心创伤
    gold_finger: string;
    gold_finger_cost: string;
  };
  arc: {
    starting_state: string;      // 开篇状态
    pressure_choice: string;     // 关键压力选择
    ending_state: string;        // 结局状态
  };
  relationships: {
    target: string;              // 关系对象
    type: string;                // 关系类型（朋友/仇人/师徒）
    tension: string;             // 关系张力
  }[];
}[]
```

## 完成标准

- 主角的 conscious_desire / unconscious_need / fatal_flaw 三项 confidence ≥ high
- 至少 1 个反派有完整 4 层结构
- 金手指必须有代价

---

## 推荐问题

1. "我想添加一个角色" → 进入 ③ 角色深潜
2. "我的主角有点单薄" → 直接追问 unconscious_need
3. "我的反派很无聊" → 追问反派的 3 个必备
4. "金手指怎么设计" → 引导代价设计
5. "我想写一个配角" → 询问 Tier 层级，决定追问深度

## 短篇视角：1-3 人 + 能否砍掉

> **触发条件**：length_type === "short"，进入 ③ 模块时。

短篇复用 ③ 角色深潜 5 层追问链，但**追问密度降低、必加额外检查**。

**短篇专属约束**：

1. **角色数 ≤ 3 人**
   - 主角 1 人 + 配角 0-2 人
   - 超过 3 人 → 触发"能否砍掉"检查

2. **"能否砍掉"检查（必做）**
   - 问：这个角色从故事里删掉，主线会变吗？
   - 答"不会" → 这个角色没用 → 删
   - 答"会" → 保留，但追问"为什么必须存在"

3. **一句话人设锚点（必填）**
   - 模板：[姓名] 是 [身份]，但 [反差/秘密]。他最想要 [欲望]，但 [致命弱点] 让他 [困境]。
   - 短篇没有空间铺人设，必须 1 句话让读者记住

4. **对手可以是抽象的**
   - 不一定是人：时间、社会、命运、规则、记忆
   - 短篇不一定要"反派"——可以是"困境"

5. **金手指可有可无**
   - 长篇金手指是核心爽点来源
   - 短篇可有可无（情绪驱动 > 设定驱动）
   - 如果有，必须有代价（沿用长篇约束）

**苏格拉底引导（短篇版）**：
- 问 1：你的故事有几个角色？能砍到 3 个以内吗？
- 问 2：每个角色"砍掉会怎样"？
- 问 3：主角能用 1 句话描述吗？（一句话人设公式）
- 问 4：短篇需要"反派"吗？还是"困境"就够？
- 问 5：5 层深挖只走 3 层（表面/动机/代价）——这是字数经济选择
