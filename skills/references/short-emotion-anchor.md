# 情绪锚点 · 短篇专属方法论

> 在主 Prompt 中识别用户进入"短篇路径"且提到「情绪钩子/主导情绪/导语」时，AI 应加载并应用本文档。
> 替代长篇 ① 选题锚定。

---

## 模块目标

短篇不是"长篇的缩短版"。在动笔之前，必须先**锁定一个主导情绪**，让所有元素（角色、情节、意象、结尾）都服务于这个情绪。这是爱伦·坡在《短篇小说论》中提出的"单一效果"（Unity of Effect）——短篇小说的最高法则。

---

## 5 层追问链（短篇版）

> 短篇受字数经济约束，可走 3 层压缩版（表面/动机/代价）或 5 层完整版（每层 1 问即停）。

### 第 1 层：主导情绪是什么？

**问题模板**：
- 读者读完你这篇短篇，**应该带走一种什么感觉**？
- 用 1 个词或 1 个短语描述这个情绪
- 候选：压抑、恐惧、温暖、孤独、荒诞、悬疑、惊喜、悲伤、希望、绝望、治愈、惊悚

**判断标准**：
- ✅ 「压抑中的希望」（具体、复合、有层次）
- ❌ 「好看」（太泛）
- ❌ 「爽」（太单维）

**关键测试**：作者能用 1 个复合短语说出这个情绪，而不是 1 个单词。

### 第 2 层：情绪的根是什么？

**问题模板**：
- 为什么是这个情绪？什么触动了你？
- 你生活中有过类似的感受吗？什么时候？
- 短篇的"作者连接"比长篇更深——情绪是你自己体验过的

**关键洞察**：最打动人心的短篇，情绪一定来自作者的亲身体验或强烈共鸣。

### 第 3 层：情绪如何被验证？

**问题模板**：
- 写完之后，怎么判断情绪传达成功了？
- 你愿意让陌生人读这篇，然后问"你感受到了什么"吗？
- 如果对方说出的情绪不是你说的——是写错了还是写偏了？

**关键测试**：作者能想出至少 1 个具体场景/画面，能让读者产生这个情绪。

### 第 4 层：情绪的边界在哪？

**问题模板**：
- 这个情绪会不会"过"？读者会不会觉得太重/太轻/太黏？
- 你希望读者读完之后是"想哭"还是"沉默"还是"释然"？
- 这个情绪和你选择的载体（悬疑/情感/科幻）匹配吗？

**反模式警告**：
- 情绪和类型不匹配：「治愈系短篇但情绪是'绝望'」→ 割裂
- 情绪过度：「想让读者哭但写得像祥林嫂」→ 反弹

### 第 5 层：情绪会变吗？

**问题模板**：
- 故事结尾，**情绪是同一个情绪的更深处**，还是**翻转成另一个情绪**？
- 例：开篇是"孤独"，结尾是"孤独中的自我接纳"——同情绪深化
- 例：开篇是"压抑"，结尾是"压抑后的释放"——情绪翻转

**关键测试**：作者能用一句话说清"情绪从 X 变成 Y"。

## 关键方法论

### 单一效果理论（爱伦·坡）

> "短篇小说家应该让他的每一句话都服务于一个预设的情绪目标。读者从头读到尾，应该只感受到一种情绪的渐变。"

**核心约束**：
- 一个故事 = 一个主导情绪
- 所有元素（人物、情节、意象、对话、节奏）都服务于这个情绪
- 任何与情绪无关的元素都应该被删掉

### 主导情绪判定流程

```
1. 作者先说"我想写一个 ___ 的故事"
   → 提取情绪词
   
2. 测试这个情绪词：
   - 1 个词能说出吗？→ 太单维，需细化
   - 1 个复合短语能说出吗？→ 可能是主导情绪
   - 1 段话能描述吗？→ 可能是"主题"而非"情绪"，需重聚焦

3. 用 3-5 个具体画面测试：
   - 画面 A：能否引发这个情绪？
   - 画面 B：能否引发这个情绪？
   - 画面 C：能否引发这个情绪？
   - 如果只有 1 个画面能 → 情绪太窄，需扩展
   - 如果 3 个都能 → 找到了

4. 对照反例：作者能说出"这不是恐惧而是 X"——情绪才清晰
```

### 一句话钩子（Hook One-Liner）

> 短篇的钩子必须能在 1 句话内把读者拉进去。

**6 种钩子原型**：

| 类型 | 模板 | 例子 |
|------|------|------|
| 反常 | "所有人都在做 X，但我在做 Y" | 所有人都在逃跑，只有我朝火里走去 |
| 揭示 | "我从未想过，X 会在 Y 时发生" | 我从未想过，自己会在 24 岁生日那天杀掉最好的朋友 |
| 预言 | "所有人说我会成为 A，但我知道我会是 B" | 所有人都说我会成为英雄，但只有我知道我将成为怪物 |
| 决定 | "今天，我决定 X" | 今天，我决定结束这一切 |
| 发现 | "那 X 上，写着 Y" | 那封信上，写着我已经死了三年的名字 |
| 反差 | "X 是 Y，但 Z" | 外卖小哥单手接住了那颗子弹 |

**关键测试**：作者能用 1 句话（≤30 字）写出钩子，且能"念给朋友听让他想继续问"。

### 导语 200-500 字框架

> 短篇网文（知乎/番茄/小红书）的导语是独立产物，不是"前三章的缩减版"。

**导语必须完成的 3 件事**：

1. **抛出核心冲突**（≤100 字）
   - 主角面临什么困境？一句话说清
   
2. **设定反常情境**（≤150 字）
   - 这个世界/这个人有什么"不对劲"？
   - 创造悬念：不解释为什么不对劲
   
3. **展现悬念钩子**（≤150 字）
   - 留一个让读者必须看下去的问题
   - 钩子 = 情绪的引爆点

**导语模板**：

```
[反常情境开场]（50-80 字）
  ↓
[主角登场 + 核心冲突]（80-150 字）
  ↓
[第一个情绪爆发点]（50-100 字）
  ↓
[钩子：留一个没解的问题]（30-50 字）
```

**反模式**：
- 500 字铺世界观 → 删，读者不需要
- 主角先吃饭/先介绍家庭 → 删，直接进冲突
- "这是一个关于 X 的故事" → 删，这是元叙述
- 没有钩子直接结束 → 删，钩子是导语生命线

## 关键约束

### 短篇的情绪粒度

> 短篇情绪比长篇精细 10 倍。

| 粒度 | 例子 | 适用 |
|------|------|------|
| 大类 | 恐惧、希望、爱 | 太粗，落到长篇 |
| 中类 | 压抑、孤独、温暖 | 短篇起点 |
| 复合 | 压抑中的希望、温暖的恐惧 | 短篇标准 |
| 细腻 | 父子重逢前的最后沉默 | 短篇极致 |

**判断标准**：作者能用 1 个 5-10 字的复合短语说出情绪，且这个短语不与"市面上其他 1000 篇"重复。

### 情绪 vs 主题

| 维度 | 情绪 | 主题 |
|------|------|------|
| 定义 | 读者读完的感受 | 故事想传达的想法 |
| 表达 | "压抑中的希望" | "选择比天赋重要" |
| 关系 | 主题必须服务情绪 | 情绪可以独立存在 |
| 测试 | 读者能"感受到" | 读者能"总结出" |

**关键洞察**：短篇的"主题"必须能转化为"情绪"，否则主题就是空的。

### 情绪一致性检查

```
□ 每个角色服务于这个情绪吗？
  → 不能服务的 → 删
  
□ 每个情节事件服务于这个情绪吗？
  → 不能服务的 → 删
  
□ 每个意象服务于这个情绪吗？
  → 不能服务的 → 删
  
□ 对话服务于这个情绪吗？
  → 不能服务的 → 删

□ 节奏服务于这个情绪吗？
  → 节奏紧 = 情绪紧，节奏缓 = 情绪缓
```

## SB 字段定义（emotion）

```typescript
emotion: {
  unity_of_effect: string;        // 主导情绪（5-10 字复合短语）
  hook_one_liner: string;         // 一句话钩子（≤30 字）
  genre: string;                  // 类型（与情绪匹配）
  opening_200_500: string;        // 导语设计（200-500 字）
  imagery_set: string[];          // 2-3 个核心意象
  emotion_evolution: string;      // 情绪演化（"从 X 到 Y"）
  emotion_anti_target: string;    // 反例：这个情绪不是 ___ （避免混淆）
}
```

---

## 完成标准

- unity_of_effect 是 5-10 字复合短语，confidence = high
- hook_one_liner ≤ 30 字，陌生人听了一定想继续
- opening_200_500 包含 3 个要素（冲突/反常/钩子）
- emotion_anti_target 已明确（避免和常见情绪混淆）
- 至少 3 个具体画面能引发这个情绪（已测试）

---

## 反模式警告

| 反模式 | 触发信号 | 警告语 |
|--------|---------|--------|
| 情绪太单维 | "我想写个感人的故事" | "感人"是主题不是情绪；试着用 5-10 字的复合短语 |
| 情绪不匹配类型 | "我想写个悬疑但情绪是温暖" | 类型和情绪要匹配；温暖在悬疑里是反讽工具 |
| 情绪过度黏腻 | 作者反复强调"让人哭" | 短篇打动人靠克制不是渲染；冰山理论 |
| 情绪和情节脱节 | 情绪是"孤独"但故事讲的是冒险 | 情节必须服务和表达情绪 |
| 元叙述 | "这是一个关于 X 的故事" | 不要告诉读者主题；用情节让读者自己感受到 |
| 情绪模糊 | "我也说不清什么感觉" | 用 UNBLOCK：给 5 个候选情绪让作者选 |

---

## 推荐问题

1. "我想写个短篇" → 进入 ① 情绪锚点
2. "这篇的情绪是什么" → 直接进入第 1 层追问
3. "我开头不知道怎么写" → 进入导语设计
4. "钩子怎么写" → 6 种钩子原型选项
5. "我觉得情绪不对" → 触发反模式警告

## 推荐读物（供作者延伸阅读）

- 爱伦·坡《短篇小说论》（1842）—— 单一效果理论原典
- 雪莉·杰克逊《摸彩》—— 短篇情绪控制范本
- 卡佛《当我们谈论爱情时我们在谈论什么》—— 极简主义短篇
- 鲁迅《孔乙己》—— 千字短篇的情绪密度典范
