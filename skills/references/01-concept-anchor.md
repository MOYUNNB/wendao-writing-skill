# 选题锚定 · 深度方法论

> 在主 Prompt 中提到「选题锚定」时，AI 应加载并应用本文档。

---

## 模块目标

把模糊脑洞变成可验证的选题。

## 5 层追问链（详细版）

### 第 1 层（表面）：一句话能说清吗？

**问题模板**：
- 你的故事一句话能说清吗？
- 用「主角+目标+阻碍」三要素讲一遍

**判断标准**：
- ✅ 「废柴少年偶获传承，在修仙界逆天改命」
- ❌ 「我想写一个关于友情和成长的故事」

**如果含糊**：UNBLOCK，给 3 个选择题（"A. 修仙 B. 都市 C. 悬疑 哪个最接近？"）

### 第 2 层（动机）：为什么非写不可？

**问题模板**：
- 为什么这个故事非写不可？
- 你的独特体验/洞察是什么？
- 你有想通过这个故事传达的核心想法吗？

**关键洞察**：选题的"作者连接"是后期读者情感共鸣的种子。

### 第 3 层（矛盾）：与同类书有什么不同？

**问题模板**：
- 这个选题和当前市场的同类书有什么本质不同？
- 读者为什么选你的书而不是其他 100 本？

**反模式警告**：
- "和 XXX 差不多" → 必须问出差异点
- 没有差异点 → 询问"差异化"或推迟

### 第 4 层（极限）：值得讲一辈子吗？

**问题模板**：
- 如果主角永远无法达成目标，这个故事还值得讲吗？
- 这个故事的"核心张力"是什么？

**关键测试**：作者必须能用一句话说清"为什么这个故事值得存在"

### 第 5 层（代价）：愿意付出什么？

**问题模板**：
- 主角为了目标愿意付出什么？
- 最不想放弃的是什么？
- 这个代价足够大吗？

## 关键方法论

### 一句话法则
- 三要素：主角 + 目标 + 阻碍
- 长度：20-50 字
- 测试：念给陌生人听，看对方能否复述

### 选题四要素
| 要素 | 问题 | 通过标志 |
|------|------|---------|
| 新颖性 | 同类书多吗？ | 不超过 100 本相似设定 |
| 爽点明确 | 读者追什么？ | 能说出 3 个具体爽点 |
| 可持续性 | 能写 100 万字吗？ | 核心矛盾能裂变出多个层次 |
| 市场价值 | 谁会读？ | 目标读者群清晰 |

### Controlling Idea（McKee）
- Value + Cause：故事的核心价值（如"自由 vs 奴役"）+ 原因
- 示例：「自由会因为放弃权力而获得」
- 测试：作者能用一句话说清价值 + 原因

## SB 字段定义（concept）

```typescript
concept: {
  one_liner: string;             // 必填，一句话法则
  genre: string;                 // 必填，类型（玄幻/都市/悬疑等）
  core_appeal: string;           // 核心爽点
  market_differentiator: string; // 市场差异
  controlling_idea: string;      // McKee Value + Cause
  author_connection: string;     // 作者连接
}
```

---

## 完成标准

- one_liner / genre / controlling_idea 三项 confidence ≥ high
- 作者能用一句话对外介绍这个故事
- 触发 GRADUATE，进入模块 ②

---

## 反模式警告

| 反模式 | 触发信号 | 警告语 |
|--------|---------|--------|
| 大而全选题 | "我想写一部史诗" | 史诗是结果不是起点；先聚焦一个核心矛盾 |
| 无差异选题 | "和 XXX 差不多" | 没有差异，读者不会选你 |
| 自嗨型选题 | "我就是想写" | 创作自由 ≠ 闭门造车；考虑读者 |
| 题材小众到无受众 | "这个类型没人写过" | 可能是蓝海也可能是死海；做调研 |

## 推荐问题（开场用）

1. "我想开一本新书" → 进入 ① 选题锚定
2. "我有个模糊的脑洞" → 先让作者说出来，识别属于哪个模块
3. "我的选题卡住了" → 直接进入第 2 层追问
4. "我想换一本新书" → 询问新选题，与旧书是否相关

## 短篇视角：情绪锚点

> **触发条件**：用户在开场白说"短篇"/"千字文"/"知乎体"等，或 length_type === "short"。

短篇不走"选题锚定"路径。**替代模块是 ① 情绪锚点**（详见 `references/short-emotion-anchor.md`）。

核心差异：
- 不问"一句话法则"，问"主导情绪是什么"（Unity of Effect）
- 不做"市场差异化"，做"情绪复合短语"（如"压抑中的希望"）
- 不做"100 万字可持续性"，做"200-500 字导语设计"
- 不做 5 层深挖，可压缩为 3 层（表面/动机/代价）

**短篇触发词**：
- 直接："写短篇"、"短故事"、"千字文"、"微小说"
- 平台："知乎体"、"番茄短篇"、"小红书短篇"
- 暗示："想写个短的"、"这篇 5000 字"、"投短篇"
